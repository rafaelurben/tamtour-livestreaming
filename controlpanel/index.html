OBS Einstellungen

Um OBS zu verbinden, muss die OBS Websocket API aktiviert sein. Diese kann in OBS via Tools -> Websocket Server Settings aktiviert werden. Es ist empfohlen, ein Passwort zu setzen.

ATEM Einstellungen

Die Verbindung mit dem ATEM Mini Pro erfolgt via openswitcher-proxy. Eine modifizierte .exe kann hier heruntergeladen werden.

Startlisten-Import

Hier kannst du Startlisten importieren. Die Startlisten müssen im JSON-Format vorliegen.

KEY1 Flying Key Optionen

Hier kannst du die Position des flying keys einstellen. Der Koordinatenursprung liegt unten links und geht von 0 bis 1. Mit dem Ankerpunkt kannst du festlegen, auf welche Ecke des Bildes sich die Koordinaten beziehen.
